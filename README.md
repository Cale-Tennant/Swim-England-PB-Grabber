This project is a brief python script that grabs swim england PB data using only a membership code from the swim england website https://www.swimmingresults.org/individualbest/ and allows the user to sort and search through them

How To Run:
-Download python
-Download main.py
-Download requirements.txt
-Install required libraries, in terminal run 'pip install -r requirements.txt'
-Run main.py

How To Use:
-Follow Instructions

To find out someones swim england membership number just go to https://www.swimmingresults.org/individualbest/ and find them and then there membership number will be at the top.
